Ficha Catalográfica (Scriptcase Based)

Este projeto é parte de um gerador de fichas catalográficas. O gerador foi desenvolvido utilizando Scriptcase, uma ferramenta low-code para aplicações web em PHP. 


⚡ Funcionalidades

Geração automática de fichas catalográficas

Formulário para inserir dados do documento

Exportação em PDF

Layout compatível com padrões acadêmicos


Tela de funcionamento 

![Formulário da Ficha Catalográfica](img_ficha/ficha_catalografica_formulario.png)
![Formulário - Tela 2](img_ficha/ficha_formulario_2.png)

A partir desse formulário, o aluno irá preencher os dados de seu Trabalho de Conclusão de Curso.

Ao preencher as informações e solicitar a ficha catalográfica, os dados preenchidos serão enviados para uma Inteligência Artificial (IA), responsável por atribuir um valor de CDU ( Classificação Decimal Universal, um sistema internacional de organização de bibliotecas que categoriza o conhecimento em 10 classes principais utilizando números arábicos.). O valor atribuído é retornado então ao aluno via pdf, nos moldes da ficha catalográfica da instituição. 

O PDF está pronto então para o aluno inserir no seu trabalho acadêmico.

O sistema ainda registra todas as fichas geradas, dessa forma há um controle maior da quantidade de fichas feitas, bem como as informações que foram colocadas pelo aluno.
